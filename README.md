## 功能

- **单张处理**：上传图片，选取任意正方形区域，自动切分为九宫格，模拟朋友圈/QQ 空间预览
- **批量处理**：多张图片独立裁剪，支持单张/全部打包下载
- **间距调整**：自定义九宫格内部间距（1~10px）
- **背景颜色**：自定义九宫格区域背景色，实时预览
- **历史记录**：处理记录存储在浏览器本地（IndexedDB），可预览/下载/删除
- **隐私安全**：所有处理在客户端完成，不上传任何图片到服务器

## 快速开始

无需安装任何依赖，直接访问即可使用。

```
# 本地运行
直接打开index.html
或访问 https://tool.flowersink.com/nine-grid
```

## 技术栈

- **前端**：原生 HTML/CSS/JS，无框架
- **裁剪引擎**：Cropper.js
- **打包下载**：JSZip + FileSaver.js
- **浏览器存储**：IndexedDB（历史记录）

## 使用说明

1. 在"单张处理"标签页上传图片，裁剪区域默认选中整张图的正方形区域
2. 可拖动/缩放裁剪框，点击"生成"重新生成九宫格
3. 在右侧预览中调整背景颜色和间距
4. 点击"下载 ZIP"打包下载九张拼图
5. "批量处理"支持多张图片同时处理
6. "历史队列"自动记录所有操作，随时查看
